Instructions of use

Instructions for using the stock analysis app "Financial Analysis System" with login, categories, and filters are as follows:

  1. User Registration and Login:
    • You can log in to your account by providing your username and password. However, if you don't have one, you can create it.
    • If you haven't registered, then you can create an account by providing your username and creating a password.
    • These options are available in the upper right part of the web page at a side of the welcoming message.
  2. Dashboard:
    • Upon logging in, you will be directed to the dashboard.
    • The dashboard displays an overview of your stock portfolio, performance, and relevant financial data.
    • Filters are at the side of the colum' names. You can classify in the upper or lower criteria.
    • Financial data from your stock portfolio can be updated clicking the "Update data from tags/s" button.
    • New stocks could be added clicking the "Create new tag/s" button, then submiting the stock symbol of your preference and that's it. Some stock symbols can be found at this address:.
    • Stocks could be deleted selecting the check marks at the side of the tag's symbols on the table and then clicking the "Delete tag/s" button.
  3. Stock Data Analysis:
    • Select a stock from the category list or use the search feature to find a specific stock.
    • The app fetches stock data using the yfinance library and displays relevant information such as historical prices, technical indicators, and moving averages.
    • Apply filters to narrow down the stock list based on criteria like price range, market capitalization, or dividend yield.
    • Navigate to the "Categories" section to access different stock categories. You can create new ones selecting the check marks at the side of the tag's symbols on the table and then clicking the "Press this button to visualize categories" button.
    • Consider deleting the categories from the "⌫" button at the side of the category name.
    • Notice that you can add new stocks to a category selecting the check marks at the side of the tag's symbols on the table and then clicking the "Add tag/s to a category" button.
    • Transfer stocks between categories clicking the "Transfer tag/s between categories" button, selecting the check marks at the side of the tag's symbols on the table, clicking on the categories and then clicking on "Submit" button.
  4. Logout:
    • Safely log out from the app after use to protect your account and data. You can signout click in the upper right part of the web page at a side of the welcoming message.

Please note that this is an early version. The actual app's interface and functionalities could present some issues. Please report them if you notice this, I'm always open to suggestions. Thank you so much!

What are the functionalities of this web page?

A web page that uses the Django framework and the yfinance library could be a financial data analysis application. Let's call it "Financial Analysis System". The main features of this web page are as follows:

  • User Authentication: The web page can have user registration and login functionalities using Django's built-in authentication system. This allows users to create accounts and securely access their personalized data and settings.
  • Dashboard: Once logged in, users are presented with a personalized dashboard. The dashboard displays an overview of their financial data, such as updated stock information and other relevant functionalities.
  • Stock Data Analysis: The web page can allow users to search for and analyze stock data using the yfinance library. Users can enter the stock symbol, and the web page will retrieve the stock's historical price data, perform calculations, and generate visualizations like stock price indexes, fliters for numeric values, and technical indicators.
  • Portfolio Management: Users can manage their investment portfolio, add or remove stocks of personalized categories, and track their performance over time. The web page can show the allocation of different stocks.

The combination of Django's robust web development capabilities and the yfinance library's financial data retrieval and analysis features provides a powerful platform for creating a user-friendly and interactive financial data analysis web page. Other libraries such as json, ast, datetime, jinja2 and crispy-tailwind were used in less manner to make everything work fine. All rights reserved to their creators.
